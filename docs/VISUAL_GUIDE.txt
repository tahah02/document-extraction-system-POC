🔄 PROCESSING PIPELINE
═══════════════════════════════════════════════════════════════════════════════

User Upload
    ↓
[API: POST /upload]
    ↓
[PDF Processor] ──→ Convert PDF to images
    ↓
[OCR Engine] ──→ Extract text (PaddleOCR/EasyOCR/Tesseract)
    ↓
[Document Classifier] ──→ Identify: Payslip or Bank Statement
    ↓
[Text Cleaner] ──→ Normalize and clean text
    ↓
[Field Extractor] ──→ Extract specific fields
    ↓
[Data Validator] ──→ Validate & calculate confidence score
    ↓
[JSON Output] ──→ Save results
    ↓
User Retrieval [GET /result/{id}]


🚀 QUICK START (5 STEPS)
═══════════════════════════════════════════════════════════════════════════════

1️⃣  Navigate to project
    $ cd document-extraction-poc

2️⃣  Create virtual environment
    $ python -m venv venv
    $ venv\Scripts\activate  (Windows)
    $ source venv/bin/activate  (Linux/Mac)

3️⃣  Install dependencies
    $ pip install -r requirements.txt

4️⃣  Create directories
    $ mkdir -p uploads/raw uploads/processed output/json output/logs temp

5️⃣  Run application
    $ python -m app.main

    ✅ API available at: http://localhost:8000
