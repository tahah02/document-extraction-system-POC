================================================================================
KEY FILES DESCRIPTION
================================================================================

APPLICATION ENTRY POINT
  app/main.py
    - FastAPI application initialization
    - Server configuration (host, port, workers)
    - Background task setup
    - CORS configuration

API ROUTES
  app/api/routes.py
    - POST /api/upload              Upload PDF for processing
    - GET /api/status/{upload_id}   Check processing status
    - GET /api/result/{upload_id}   Retrieve extraction results
    - GET /health                   Health check endpoint

CORE PROCESSING
  core/pipeline.py
    - Orchestrates entire processing workflow
    - Manages PDF → Image → OCR → Extraction → Validation

  core/ocr_engine.py
    - Tesseract OCR integration
    - Text extraction from images
    - Language configuration

  core/document_classifier.py
    - Identifies document type (Payslip/Bank Statement)
    - Keyword-based classification
    - Confidence scoring

  core/extractor.py
    - Extracts specific fields from text
    - Pattern matching for field values
    - Handles both Payslip and Bank Statement formats

  core/validators.py
    - Validates extracted data format
    - Calculates confidence scores
    - Checks data completeness

DATA MODELS
  models/payslip.py
    - Fields: name, id_number, gross_income, net_income, total_deduction, month_year

  models/bank_statement.py
    - Fields: account_holder_name, account_number, statement_date

  models/extraction_result.py
    - Wrapper for API responses
    - Contains status, upload_id, results, metadata

UTILITIES
  utils/pdf_processor.py
    - Converts PDF pages to PNG images
    - Handles multi-page PDFs
    - Image quality optimization

  utils/text_cleaner.py
    - Removes special characters
    - Normalizes whitespace
    - Handles encoding issues

  utils/file_manager.py
    - Creates/manages upload directories
    - Handles file cleanup
    - Manages file paths

  utils/logger.py
    - Configures logging
    - File and console handlers
    - Log rotation

CONFIGURATION
  config/app_config.yaml
    - Server settings (host, port, workers)
    - File upload limits
    - Processing timeouts
    - Logging levels

  config/extraction_config.json
    - Field patterns for Payslip
    - Field patterns for Bank Statement
    - Keyword lists for classification

  config/ocr_config.json
    - OCR engine selection
    - Language settings
    - Confidence thresholds

================================================================================
PROCESSING PIPELINE
================================================================================

1. USER UPLOAD
   └─ POST /api/upload with PDF file

2. FILE STORAGE
   └─ Save to uploads/raw/{upload_id}.pdf

3. PDF PROCESSING
   └─ Convert PDF pages to images
   └─ Save to uploads/processed/{upload_id}/page_*.png

4. OCR EXTRACTION
   └─ Extract text from each image using Tesseract
   └─ Combine text from all pages

5. DOCUMENT CLASSIFICATION
   └─ Analyze text for keywords
   └─ Determine: Payslip or Bank Statement

6. FIELD EXTRACTION
   └─ Apply regex patterns to extract fields
   └─ Handle multiple formats and variations

7. DATA VALIDATION
   └─ Validate field formats
   └─ Calculate confidence scores
   └─ Check data completeness

8. RESULT STORAGE
   └─ Save to output/json/{upload_id}.json
   └─ Store in memory for API retrieval

9. USER RETRIEVAL
   └─ GET /api/result/{upload_id}
   └─ Return extracted data with metadata

app_config.yaml

    host: 0.0.0.0
    port: 8003
    workers: 1
  upload:
    max_file_size: 50MB
    allowed_formats: [pdf, jpg, png]
  processing:
    timeout: 300 seconds
    temp_dir: ./temp

extraction_config.json
  payslip:
    keywords: ["Gaji", "KWSP", "Cukai Pendapatan"]
    fields:
      name: ["Nama", "Name"]
      id_number: ["No K/P", "NRIC"]
      gross_income: ["Gaji Pokok", "Gross"]

      total_deduction: ["Jumlah Potongan", "Deduction"]
      month_year: ["Bulan", "Month"]

ocr_config.json
  engine: tesseract
  language: mal+eng
  confidence_threshold: 0.5
  image_preprocessing: true

================================================================================
API ENDPOINTS SUMMARY
================================================================================

POST /api/upload
  Description: Upload PDF for extraction
  Request: multipart/form-data with file
  Response: {status, upload_id, message}
  Example: curl -F "file=@document.pdf" http://localhost:8003/api/upload

GET /api/status/{upload_id}
  Description: Check processing status
  Response: {status, upload_id, message}
  Example: curl http://localhost:8003/api/status/abc-123-def

GET /api/result/{upload_id}
  Description: Get extraction results
  Response: {upload_id, file_type, documents, summary, metadata}
  Example: curl http://localhost:8003/api/result/abc-123-def

GET /health
  Description: Health check
  Response: {status: "healthy"}
  Example: curl http://localhost:8003/health

================================================================================
TROUBLESHOOTING
================================================================================

Tesseract Not Found
  → Install from: https://github.com/UB-Mannheim/tesseract/wiki
  → Set TESSERACT_CMD in app/main.py

oneDNN Warning
  → Environment variables already configured in app/main.py
  → Safe to ignore

Low Confidence Scores
  → Improve extraction patterns in core/extractor.py
  → Adjust OCR settings in config/ocr_config.json
  → Test with better quality PDFs

Processing Timeout
  → Increase timeout in config/app_config.yaml
  → Check file size (max 50MB)
  → Verify OCR engine is working
