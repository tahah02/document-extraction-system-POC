================================================================================
PROCESSING PIPELINE
================================================================================

The processing pipeline is fully implemented with the following steps:

1. User uploads PDF
2. File saved to uploads/raw/{upload_id}/
3. PDF converted to images
4. Text extracted using OCR
5. Document type classified
6. Fields extracted based on type
7. Data validated and scored
8. Language detected
9. Results saved to output/json/{upload_id}.json
10. User retrieves results via API

Average Processing Time: 15-30 seconds per PDF

================================================================================
API ENDPOINTS
================================================================================

POST /api/upload
- Upload PDF for processing
- Returns: {status, upload_id, message}

GET /api/status/{upload_id}
- Check processing status
- Returns: {status, upload_id, message, detected_language, language_confidence}

GET /api/result/{upload_id}
- Get extraction results
- Returns: {upload_id, file_type, documents, summary, detected_language, ...}

GET /health
- Health check
- Returns: {status, version}

Bank Statement (3 fields):
1. account_holder_name - Account owner
2. account_number - Bank account number

================================================================================
TROUBLESHOOTING
================================================================================

Server Won't Start:
- Check port 8004 is available
- Verify Python 3.13+ installed
- Check all dependencies installed

Low Confidence Scores:
- Improve extraction patterns in core/extractor.py
- Adjust OCR settings in config/ocr_config.json
- Use higher quality PDFs

Processing Timeout:
- Increase timeout in core/pipeline.py
- Check file size (max 50MB)
- Verify OCR engine is working

Tesseract Not Found:
- Install from: https://github.com/UB-Mannheim/tesseract/wiki
- Set path in app/main.py line 15
